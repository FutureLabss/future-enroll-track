# FutureLabs Invoicing & Enrollment System

A standalone invoicing and enrollment management system for FutureLabs programs.

## Project info

This project handles student enrollment, invoicing, payment tracking, and partner organization sponsorships.

## How can I edit this code?

**Use your preferred IDE**

Clone the repo and push changes. Make sure Node.js & npm are installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate into the project
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the dev server
npm run dev
```

**Edit a file directly in GitHub**

- Navigate to the desired file(s)
- Click the "Edit" (pencil) button at the top right of the file view
- Make your changes and commit

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (database, auth, edge functions, storage)
